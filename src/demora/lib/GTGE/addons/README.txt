=============================================
Golden T Game Engine (GTGE) Add-Ons / Plugins
=============================================


What is it?
-----------

GTGE Add-Ons/Plugins is a pack of add-ons to enhance the basic GTGE functionality.

These add-ons are not inserted to the GTGE library (GTGE_x_x_x.jar) because of :
- the classes is dependent on external library;
- the classes is not essential in the game development.

By removing any dependencies to external 3rd party library, 
GTGE would be compact and can run only with the standard Java bundle (J2SE).


Requirements
------------

- GTGE library (GTGE_x_x_x.jar)
  http://www.goldenstudios.or.id/products/GTGE/



Unzipped Files
--------------

GTGE_add_ons.jar  : GTGE add-ons library
README.txt        : this readme
[docs]            : API documentation
[src]             : source Code



Installation
------------

Simply include GTGE library (golden_x_x_x.jar), GTGE add-ons library (GTGE_add_ons.jar), 
and any required external 3rd party libraries into your build process (compile and run).
Please refer to GTGE add-ons API documentations for the required external 3rd party libraries.




How To Use
----------

Please refer to the API documentations [docs]/index.html for how to use the add-ons on your game.
