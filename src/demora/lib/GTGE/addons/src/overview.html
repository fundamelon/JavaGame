Golden T Game Engine (GTGE) Add-Ons / Plugins API Documentation,
Used along with GTGE Frame Work to enhance the basic GTGE functionality.

GTGE is available to download at http://www.goldenstudios.or.id/products/GTGE/.

What's available in this add-ons :

  • {@linkplain com.golden.gamedev.OpenGLGameLoader OpenGL renderer} via LWJGL
  • {@linkplain com.golden.gamedev.OpenGLGameLoader OpenGL renderer} via JOGL
  • {@linkplain com.golden.gamedev.engine.audio.JavaLayerMp3Renderer MP3 audio player} using JavaLayer
  • {@linkplain com.golden.gamedev.engine.audio.JOrbisOggRenderer OGG audio player} using JOrbis
  • {@linkplain com.golden.gamedev.engine.timer.GageTimer Hi-Res Timer} based on GageTimer

Please refer to the related API documentation for how to use it on your game.
For example: Read {@linkplain com.golden.gamedev.engine.audio.JavaLayerMp3Renderer} API documentation for how to use MP3 audio player.

And be sure to read the installation procedure below first.

System Requirements

  • GTGE system requirements
  • GTGE library (golden_x_x_x.jar)
  • External 3rd party library mentioned by particular GTGE plugins API, such as:
    • JavaLayer library for playing MP3
    • LWJGL library for rendering OpenGL via LWJGL
    • GageTimer library for hi-res timer

Installation

Include GTGE add-ons library (GTGE_add_ons.jar) and any required external 3rd party library into your build process (compile and run).

For example :
Compiling and running YourGame.java with GTGE add-ons (GTGE_add_ons.jar) and JavaLayer (javalayer.jar)
Assumption: all class files and libraries are inside C:\GameProject directory and Java has been installed properly 

   C:\GameProject>javac -classpath %CLASSPATH%;.;golden_0_2_1.jar;GTGE_add_ons.jar;javalayer.jar YourGame.java
   C:\GameProject>java -classpath %CLASSPATH%;.;golden_0_2_1.jar;GTGE_add_ons.jar;javalayer.jar YourGame

For more information about classpath in installation (compile and run), please read GTGE Tutorial Chapter 2.
